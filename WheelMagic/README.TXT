【ソフト名】 WheelMagic for TTBase
【著作権者】 Copyright(c) 2009-2011 itow.y, 2014 tapetums
【種　　別】 TTBase プラグイン
【動作環境】 Windows XP/Vista/7/8/8.1
【開発環境】 Visual C++ 2013 Express Edition


【概要】
マウスのホイール操作を、マウスカーソル直下のウィンドウに適用するためのプログラムです。
TTBase もしくは 互換ソフトのプラグインとして動作します。

itow.y さん作の WheelMagic 0.2.0 を基にしています。


【アーカイブ内容】
README.TXT         このファイル
x86\WheelMagic.dll 32-bit版 バイナリ
x64\WheelMagic.dll 64-bit版 バイナリ
WheelMagic_src.zip ソースコード


【インストール】
本体となるプログラムによります。
TTBase の場合は、本体のインストールフォルダ以下の場所に配置してください。

本体が 32-bit版 の場合は x86\WheelMagic.dll を、
本体が 64-bit版 の場合は x64\WheelMagic.dll を それぞれお使いください。


【アンインストール】
DLLファイルを削除してください。設定ファイル等は使用していません。
常駐型のプラグインのため、本体を一旦終了させる必要があります。
